# Vidiony Project Brief

## Overview

Vidiony is a video sharing platform built with Next.js that enables users to upload, view, and share videos. The platform includes authentication, video playback, and social features.

## Core Requirements

- **Video Upload**: Users can upload video content
- **Video Playback**: Smooth video playback experience
- **User Authentication**: Login and registration functionality
- **Social Features**: Video sharing capabilities
- **Responsive Design**: Works across desktop and mobile devices

## Technology Stack

- **Frontend**: Next.js 14+, React, TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Package Manager**: Bun

## Key Features

1. User authentication (login/register pages)
2. Video upload functionality
3. Video playback with viewing page
4. Video card display for feeds
5. Navigation and layout components

## Current Status

- Project in early development stage
- Frontend structure established
- Core pages and components implemented
- Backend integration pending
